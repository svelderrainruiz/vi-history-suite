VI History Suite

vi-history-suite is a Visual Studio Code extension for reviewing LabVIEW VI history in Git repositories.

Use the installed guide inside the extension for install, checkbox-selected compare workflow, retained comparison review, and execution-policy tasks without needing the broader engineering control plane.

Product Promise

The current ship target is to release vi-history-suite as a versioned VSIX that a user can install and use to:

  • right-click an eligible LabVIEW VI in a trusted Git repository
  • review commit history for that VI
  • generate reliable NI comparison reports
  • review a metadata-concentrated dashboard across at least three commits
  • retain a separate human decision record without mutating machine evidence

Install Surfaces

Current install and release surfaces are:

  • preview VSIX artifact from the main pipeline
  • local package output via npm run package
  • governed tagged release artifact retained under release-evidence/
  • packaged bundled user docs opened through VI History: Open Documentation or the history-panel Open docs action
